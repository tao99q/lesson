Margin Demo

No styles are applied to either of these paragraphs.

Margin Demo—with borders turned on

This is the same arrangement of a heading and two paragraphs but now the borders are visible, and you can see how the default margins creates space between the elements.

Margins push an element away from elements it would otherwise touch.

Margin Demo—borders on, but without margins

Here the margins are set to 0 (zero), so the elements' borders now touch.

Be aware that almost every element has margins by default, and you will often want to eliminate or adjust the default margin settings to get the effect you want.
